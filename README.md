### 🔭 Overview
Jarvis v1.0 serves as the core framework for my AI development journey. It is built on a **modular Python architecture**, separating logic into distinct components to ensure clarity, maintainability, and ease of expansion.

### 🏗 Architecture
The system is divided into key modules:
* `main.py`: The entry point and main control loop.
* `brain.py`: Core logic and decision-making processing.
* `commands.py`: Command routing and task execution.
* `listen.py`: Audio input and speech-to-text processing.
* `voice.py`: Text-to-speech synthesis.
